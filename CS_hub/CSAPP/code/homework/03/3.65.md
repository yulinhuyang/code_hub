# 3.65

## A & B

从汇编代码的 6，7 行，可以看到

```text
addq $8, %rdx
addq $120, %rax
```

在每次循环中， j++ ，可知 `A[j][i]` 比 `A[i][j]` 地址变化要大。

所以， `A[i][j]` 的指针保存在 %rdx, `A[j][i]` 的指针保存在 %rax。

## C

利用 178 页公式 3.1 与汇编代码第 7 行，可得

```text
X_A + L * ( M * j + i ) + 120 = X_A + L * ( M * ( j + 1 ) ) + i )
与 L = 8，可知
M = 15
```
